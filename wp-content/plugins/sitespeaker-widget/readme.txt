Let your users listen to your content with the ReadAloud text-to-speech widget

== Description ==

The ReadAloud text-to-speech widget is embedded into your posts and give users an alternate way to consume your content as audio.

== Installation ==

1. [Sign up](https://portal.readaloudwidget.com/) for a free account.
2. After signup, login to the customer portal and add your website domain(s).
3. Click on 'Get Code' in the portal and note down your API key.
4. Install the ReadAloud Widget plugin through the WordPress plugins store.
5. Activate the plugin through the 'ReadAloud Widget' Settings menu in WordPress.

== Frequently Asked Questions ==

= What is Text to Speech(TTS) technology? =

= How does the ReadAloud widget work? = 

The ReadAloud widget appears at the top your posts as a button labeled "Listen to this article".  When clicked, the button is replaced by an audio player that plays back the synthesized speech of your article's content.

= How much does it cost? = 

We provide a free plan for customers with small websites and low traffic.  Beyond a level of usage, you will be asked to upgrade to a Pay-As-You-Go plan, which is billed based on the number of characters of text synthesized and the number of listens per month.

= How many websites can I install the widget on? = 

You can install the widget on any number of websites you own.  Billing is cumulative across all the websites for your account.

= How do my website visitors use the widget?  Do they need to install anything? = 

Users simply click on the "Listen to this article" button to start listening right inside the browser, no software download required.

= Which devices and browsers are supported? = 

All device types including desktop, mobile, and tablet devices are supported.

= What languages are supported? = 

The widget supports more than 25 languages, including English, Spanish, French, German, Hindi, Chinese Mandarin, Italian, Portuguese, Russian, Japanese, and Arabic.

= Where should the Listen button be placed on a website? = 

We recommend placing the widget at the top of articles or in a visible place above the fold in the content.

= Does ReadAloud collect or sell my website users' data? = 

No, the widget does not collect user data from the websites it is installed on.

= Can I cancel my account? = 

You can cancel any time by contacting us via the contact form on our main website.
